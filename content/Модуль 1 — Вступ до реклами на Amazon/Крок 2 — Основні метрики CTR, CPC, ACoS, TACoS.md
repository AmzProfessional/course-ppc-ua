**Мета**: Засвоїти чотири базові метрики PPC з формулами, цільовими порогами і розумінням, коли метрика сигналить про проблему. Це фундамент, без якого наступні модулі не читаються.

> [!warning] Усі формули нижче — робочі. Жодної метрики «для краси». Кожна з них використовується при прийнятті рішень у Модулях 5-6.

---

## 1. CTR — Click-Through Rate

**Формула:**
```
CTR = Clicks / Impressions × 100%
```

CTR показує, наскільки оголошення релевантне пошуковому запиту і наскільки лістинг «продає» сам по собі (фото, title, ціна, зірки).

### Цільові пороги CTR за типом відповідності і плейсментом

| Контекст | Мінімум | Хороший рівень |
|----------|---------|----------------|
| Exact Match | 1.0% | 1.5%+ |
| Phrase / Broad Match | 0.5% | 0.8%+ |
| Auto / Product Targeting (PAT) | 0.3% | 0.5%+ |
| Top of Search (ToS) | 5.0% | 7%+ |
| Rest of Search (RoS) | 1.0% | 1.5%+ |
| Product Page (PP) | 0.3% | 0.5%+ |

**CTR < 0.3% на Exact / ToS** — червоний прапор. Причини у порядку ймовірності: слабкі фото, неконкурентна ціна, низький rating, нерелевантне ключове слово, проблемний бренд. Підвищення бідів це **не** вирішує.

### Приклад
Кампанія Exact Match за тиждень: 12 000 показів, 78 кліків.
CTR = 78 / 12 000 × 100 = **0.65%** — нижче таргету 1% для Exact → перевірити лістинг.

## 2. CPC — Cost Per Click

**Формула:**
```
CPC = Spend / Clicks
```

CPC — середня фактична ціна одного кліку (не бід). Amazon списує за second-price auction, тому CPC завжди нижчий за встановлений бід.

### Як інтерпретувати

- **CPC ≈ бід** → мала конкуренція, можна знижувати бід без втрати трафіку.
- **CPC значно нижчий за бід (0.6× і нижче)** → є запас на зниження біда.
- **CPC = бід або upper limit** → жорсткий аукціон, підвищення біда дасть мало нового трафіку.
- **CPC росте без зростання продажів** → конкуренти підняли біди, треба переглядати target ACoS.

### Типові рівні CPC

Залежить від ніші і ціни товару. Орієнтовні діапазони для US-маркету:

| Ціна товару | Типовий CPC на Exact |
|-------------|----------------------|
| $10-20 | $0.40-0.90 |
| $20-50 | $0.80-1.80 |
| $50-100 | $1.50-3.50 |
| $100+ | $2.50-6.00 |

**Не правило, а бенчмарк.** У конкурентних нішах (supplements, beauty) CPC на 30-50% вище.

## 3. ACoS — Advertising Cost of Sale

**Формула:**
```
ACoS = Ad Spend / Ad Sales × 100%
```

ACoS показує, який відсоток рекламних продажів «з'їдає» сам рекламний бюджет. Чим нижчий — тим ефективніша кампанія.

### Target ACoS — що це на практиці

Простіша схема для Модуля 1:
- **Target ACoS** — ACoS, якого ми цілимо для ASIN.
- Для усталених ASIN: приблизно дорівнює маржі (Net Margin з Sellerboard).
- Для launch: тимчасово вище маржі.

> [!note] Справжнє визначення Target ACoS, з формулою `max(Margin, Break-even ACoS)`, cap 30%, floor 10% — у Модулі 5 Крок 1. Там ACoS завжди порівнюється з target, а не з якимось абстрактним «добрим» значенням.

### Пороги Real ACoS

| ACoS vs Target | Статус |
|----------------|--------|
| < Target / 1.3 | Профітний — кандидат на підсилення |
| Target / 1.3 ... Target | Оптимальна зона |
| Target ... 1.5× Target | Перевищення — плановане зниження бідів |
| 1.5× Target ... 2× Target | Серйозне перевищення — мануальний огляд |
| > 2× Target АБО 10+ кліків, 0 ордерів | Chronic bleed — пауза або переробка |

### Приклад
Spend = $184, Ad Sales = $512.
ACoS = 184 / 512 × 100 = **35.94%**.
Якщо target = 22% → 35.94 / 22 = 1.63× → серйозне перевищення, мануальний огляд + зниження бідів.

## 4. TACoS — Total Advertising Cost of Sale

**Формула:**
```
TACoS = Ad Spend / Total Sales × 100%
```

Де `Total Sales` = PPC Sales + Organic Sales.

TACoS відображає залежність бренду від реклами. Це **стратегічна метрика** — для щоденних рішень не використовується, але показує напрямок руху ASIN.

### Інтерпретація динаміки TACoS

- **TACoS падає WoW при стабільному Spend** → органіка росте, реклама «будує» OR. Найкращий сценарій.
- **TACoS стабільний** → реклама і органіка ростуть пропорційно. Нормально.
- **TACoS росте** → реклама дорожчає або органіка просідає. Тривожний сигнал — перевірити OR, відгуки, ціну.
- **TACoS > 20% тривалий час** → надмірна залежність від PPC, бренд не вибудовує органічні продажі.

### Приклад
Spend тижня = $250, PPC Sales = $900, Organic Sales = $1 850.
TACoS = 250 / (900 + 1 850) × 100 = 250 / 2 750 × 100 = **9.09%**.

## 5. Додаткові метрики

Не входять у базову четвірку, але постійно згадуються:

| Метрика | Формула | Для чого |
|---------|---------|----------|
| **CVR** (Conversion Rate) | Orders / Clicks × 100% | Оцінка якості лістингу. Таргет 8-15%. <5% — проблема |
| **CPA** (Cost per Acquisition) | Spend / Orders | Вартість одного замовлення |
| **RoAS** (Return on Ad Spend) | Ad Sales / Ad Spend | Обернений ACoS. RoAS = 1 / ACoS |
| **%PPC** у продажах | PPC Sales / Total Sales × 100% | Показує, скільки продажів генерує реклама |

### Практичний кейс

Дано дані по 4 кампаніях бренду `HomeGoods` за останні 7 днів. Target ACoS = 25%.

| Кампанія | Impressions | Clicks | Spend | Ad Sales | Type |
|----------|-------------|--------|-------|----------|------|
| SP-Exact-A | 8 000 | 112 | $140 | $620 | Exact |
| SP-Broad-B | 45 000 | 85 | $95 | $180 | Broad |
| SP-Auto-C | 22 000 | 40 | $38 | $210 | Auto |
| SP-PAT-D | 12 000 | 24 | $30 | $0 | PAT |

**Завдання:**
1. Порахуйте CTR, CPC, ACoS для кожної кампанії.
2. Яка кампанія найкраща, яка найгірша?
3. Які дії варто зробити з найгіршою?

*Розв'язок:*

| Кампанія | CTR | CPC | ACoS |
|----------|------|------|------|
| SP-Exact-A | 1.40% | $1.25 | 22.58% |
| SP-Broad-B | 0.19% | $1.12 | 52.78% |
| SP-Auto-C | 0.18% | $0.95 | 18.10% |
| SP-PAT-D | 0.20% | $1.25 | — (0 продажів) |

2. **Найкраща** — SP-Auto-C: ACoS 18% нижче target 25%, CVR хороший. Проте CTR 0.18% нижче мінімуму 0.3% для Auto — лістинг перевірити. **Найгірша** — SP-Broad-B: CTR 0.19% (нижче 0.5% для Broad), ACoS 52.78% — у 2.1× перевищує target.

3. SP-Broad-B: знизити біди на 15-20% (логіка — Модуль 5 Крок 4), перевірити Search Term Report на сміттєві запити, додати негативні. SP-PAT-D: 24 кліки, 0 ордерів — chronic bleed, мануальний огляд, кандидат на паузу.

### Розрахункова задача

ASIN B07CALC за тиждень:
- Impressions: 18 500
- Clicks: 148
- Spend: $192
- PPC Sales: $540
- Organic Sales: $1 260

**Порахуйте:**
1. CTR
2. CPC
3. ACoS
4. TACoS
5. %PPC у загальних продажах
6. Якщо target ACoS = 22%, у якій зоні знаходиться ACoS?

*Розв'язок:*
1. CTR = 148 / 18 500 × 100 = **0.80%**
2. CPC = 192 / 148 = **$1.30**
3. ACoS = 192 / 540 × 100 = **35.56%**
4. TACoS = 192 / (540 + 1 260) × 100 = 192 / 1 800 × 100 = **10.67%**
5. %PPC = 540 / 1 800 × 100 = **30%**
6. ACoS / target = 35.56 / 22 = 1.62× → **серйозне перевищення, мануальний огляд + зниження бідів** (зона 1.5×-2× target).
