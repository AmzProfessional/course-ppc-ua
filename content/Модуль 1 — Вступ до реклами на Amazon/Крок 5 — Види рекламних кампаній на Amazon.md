## 1. Значення різних типів кампаній

> [!note] Amazon пропонує три основні типи PPC-кампаній, кожен із яких має унікальні цілі та сценарії використання. Розуміння їх допомагає вибрати правильну стратегію для вашого магазину.

#### **Чому це важливо?**

- **Sponsored Products**: Збільшують продажі окремих товарів через видимість у пошуку.
- **Sponsored Brands**: Підвищують впізнаваність бренду та просувають групу товарів.
- **Sponsored Display**: Допомагають залучити аудиторію через ретаргетинг або таргетинг за інтересами.
- Вибір типу кампанії залежить від цілей: збільшення продажів, брендинг чи повернення клієнтів.

## 2. Огляд типів кампаній

> [!note] Кожен тип кампанії має свої особливості, налаштування та оптимальні сценарії використання.

### Основні типи кампаній:

1. **Sponsored Products**:
    - **Що це?**: Оголошення для окремих товарів, які з’являються в результатах пошуку або на сторінках товарів.
    - **Ціль**: Збільшення продажів конкретного ASIN, особливо для нових чи конкурентних товарів.
    - **Коли використовувати?**: Для запуску нових товарів, підвищення видимості або конкуренції за ключові слова.
    - **Налаштування**:
        - Вибір ключових слів (broad, phrase, exact).
        - Встановлення ставок і бюджету.
        - Таргетинг на автоматичну або ручну кампанію.
    - **Приклад**: Просування нового кухонного ножа в пошуку за запитом "kitchen knife". 
    
2. **Sponsored Brands**:
    - **Що це?**: Оголошення для просування бренду, що включають логотип, кастомний заголовок і кілька товарів.
    - **Ціль**: Підвищення впізнаваності бренду та продажів кількох товарів.
    - **Коли використовувати?**: Для брендів із кількома ASIN або для залучення лояльної аудиторії.
    - **Налаштування**:
        - Створення привабливого заголовка (наприклад, "Найкращі товари для дому від (Ваш бренд)").
        - Вибір 3–4 товарів для просування.
        - Встановлення ключових слів і бюджету.
    - **Приклад**: Реклама лінійки посуду вгорі сторінки пошуку.
    
3. **Sponsored Display**:
    - **Що це?**: Оголошення для ретаргетингу або таргетингу за інтересами, які показуються на Amazon і поза ним.
    - **Ціль**: Повернення клієнтів, які переглядали товари, або залучення нової аудиторії за інтересами.
    - **Коли використовувати?**: Для ремаркетингу або просування товарів у конкурентних нішах.
    - **Налаштування**:
        - Таргетинг за аудиторією (наприклад, користувачі, які переглядали схожі товари).
        - Вибір продуктів для реклами.
        - Встановлення бюджету та ставок.
    - **Приклад**: Ретаргетинг клієнтів, які переглядали ваш товар, але не купили.

## 3. Процес вибору та аналізу кампаній

> [!note] Вибір і аналіз типу кампанії залежить від цілей магазину. Використовуйте чек-лист для відстеження прогресу:

- [ ]  **Визначення цілей**: Вибрати тип кампанії залежно від потреб (продажі, брендинг, ретаргетинг).
- [ ]  **Налаштування кампанії**: Створити кампанію з правильними параметрами (ключові слова, бюджет, таргетинг).
- [ ]  **Аналіз ефективності**: Перевірити метрики (ACoS, CTR, Sales) для оцінки результатів.
- [ ]  **Оптимізація**: Запропонувати зміни для покращення ефективності.

### Кроки роботи:

1. **Визначте ціль кампанії**:
    - Наприклад, Sponsored Products для нового товару, Sponsored Brands для просування бренду.
2. **Створіть кампанію в Amazon Advertising**:
    - Виберіть тип кампанії (Sponsored Products, Brands, Display).
    - Налаштуйте ключові слова, таргетинг, ставки та бюджет.
3. **Перевірте ефективність**:
    - Відкрийте Campaign Manager і оцініть метрики (CTR, ACoS, Sales) за 7 днів.
    - Для Sponsored Products використовуйте **Search Term Report** для аналізу ключових слів.
    - Для Sponsored Brands перевірте кліки на заголовок і товари, а також Sponsored Brand Search Terms.
    - Для Sponsored Display оцініть Conversion Rate і аудиторію.
4. **Оптимізуйте кампанію**:
    - Додайте негативні ключові слова для Sponsored Products.
    - Покращте заголовок чи зображення, додайте негативні ключові слова для Sponsored Brands.
    - Змініть таргетинг для Sponsored Display, якщо Conversion Rate низький.

## 4. Ключові поради для PPC Account Manager’ів

- **Вибір кампанії**: Використовуйте Sponsored Products для швидких продажів, Sponsored Brands для брендингу, Sponsored Display для ретаргетингу.
- **Тестування**: Починайте з невеликого бюджету ( 5–10 доларів / день), щоб оцінити ефективність.
- **Якість лістингу**: Успіх усіх кампаній залежить від якісного лістингу (фото, опис, відгуки).
- **Регулярний аналіз**: Перевіряйте Search Term Report для Sponsored Products і аудиторію для Sponsored Display щотижня.
- **Гнучкість**: Експериментуйте з різними типами кампаній залежно від цілей магазину.
