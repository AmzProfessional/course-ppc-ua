## 1. Значення Amazon Advertising Dashboard

> [!note] Dashboard в Amazon Advertising — це основний інструмент для швидкого перегляду ключових метрик PPC-кампаній, який допомагає оцінити їх ефективність і виявити можливості для оптимізації.

**Чому це важливо?**

- Dashboard дозволяє бачити всі ключові метрики (Impressions, Clicks, Spend, ACoS) в одному місці.
- Правильне налаштування економить час і допомагає зосередитися на критичних даних.
- Щоденний моніторинг через Dashboard є основою для прийняття рішень щодо коригування кампаній.

Фільтри в дашборді Amazon Advertising дозволяють налаштувати перегляд даних. 
"Country" визначає країну цільової аудиторії, 
"Active status" показує активні кампанії, 
"Type" фільтрує за типом кампанії, 
"Targeting" — за типом таргетингу, 
"Retailer" — за роздрібним продавцем,
"Portfolio" — за портфелем кампаній. 
Також доступні фільтри за середнім часом у бюджеті ("Avg. time in budget"), бюджетом ("Budget"), станом "Out of budget", а також за кількістю показів ("Impressions").

## 2. Налаштування Dashboard

> [!note] Налаштування Dashboard включає вибір періодів і метрик, які відповідають вашим цілям аналізу.

### Кроки налаштування:

1. **Вибір періоду**:
    - Для щоденного моніторингу: Оберіть період "Вчора" або "Останні 7 днів" для оцінки короткострокових результатів.
    - Для аналізу трендів: Використовуйте "Останні 30 днів" для виявлення довгострокових закономірностей.
2. **Вибір метрик для відображення**:
    - **Impressions**: Кількість показів оголошень.
    - **Clicks**: Кількість кліків по оголошеннях.
    - **Spend**: Витрати на рекламу.
    - **Sales**: Виручка від PPC-кампаній.
    - **ACoS (Advertising Cost of Sale)**: Відсоток витрат до продажів.
    - **CTR (Click-Through Rate)**: Відсоток кліків до показів.
3. **Налаштування таблиці**:
    - Додайте вибрані метрики до Dashboard через налаштування колонок у Campaign Manager.
    - Налаштуйте сортування за ключовими метриками (наприклад, ACoS або CTR) для швидкого аналізу.

## 3. Процес роботи з Dashboard

> [!note] Робота з Dashboard складається з трьох етапів. Використовуйте чек-лист для відстеження прогресу:

- [ ]  **Налаштування Dashboard**: Вибрати період і метрики для відображення.
- [ ]  **Перегляд даних**: Оцінити ключові метрики за вибраний період.
- [ ]  **Підготовка до аналізу**: Зібрати дані для подальшого виявлення аномалій.

### Кроки роботи:

1. **Увійдіть у Amazon Advertising**.
2. **Налаштуйте Dashboard**:
    - У Campaign Manager виберіть період "Вчора".
    - Додайте метрики: Impressions, Clicks, Spend, Sales, ACoS, CTR.
3. **Перегляньте дані**:
    - Перевірте значення метрик і порівняйте їх із цільовими (наприклад, CTR >0.5%, ACoS 10–30%).
    - Зверніть увагу на кампанії з високим ACoS або низьким CTR.
4. **Підготуйтеся до аналізу**:
    - Збережіть налаштування Dashboard для щоденного використання.
    - Експортуйте дані (за потреби) для детального аналізу в Search Term Report, про це в 3 модулі.

## 4. Ключові поради для PPC Manager’ів

- **Фокус на ключових метриках**: Починайте з ACoS і CTR, щоб швидко оцінити ефективність.
- **Регулярність**: Перевіряйте Dashboard щодня, щоб бути в курсі змін.
- **Гнучкість**: Адаптуйте вибір метрик залежно від цілей (наприклад, додайте TACoS для стратегічного аналізу).
