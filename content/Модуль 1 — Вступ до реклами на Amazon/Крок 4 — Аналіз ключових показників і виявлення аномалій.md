**Мета**: Навчитися аналізувати ключові показники PPC-кампаній у Amazon Advertising Dashboard, виявляти аномалії та пропонувати рішення для їх виправлення.

---
## 1. Значення аналізу ключових показників

> [!note] Аналіз ключових показників у Dashboard дозволяє швидко оцінити ефективність PPC-кампаній, виявити аномалії (неефективні кампанії чи ключові слова) і запропонувати дії для покращення результатів.

**Чому це важливо?**

- Виявлення аномалій (наприклад, високий ACoS або низький CTR) допомагає уникнути марних витрат.
- Аналіз причин аномалій дозволяє зрозуміти, чи проблема у кампанії, лістингу чи конкуренції.
- Пропозиції на основі аналізу підвищують рентабельність і продажі.

## 2. Підготовка до аналізу

> [!note] Ефективний аналіз починається з правильного використання Dashboard і сортування даних.

### Кроки підготовки:

1. **Відкрийте Amazon Advertising Dashboard**:
    - Переконайтеся, що Dashboard налаштовано з метриками: Impressions, Clicks, Spend, Sales, ACoS, CTR.
2. **Вибір періоду**:
    - Для щоденного аналізу: "Вчора".
    - Для трендів: "Останні 7 днів" або "30 днів".
3. **Сортування даних**:
    - Сортуйте таблицю за ACoS, CTR, Spend, щоб швидко знайти проблемні кампанії.

## 3. Процес аналізу та виявлення аномалій

> [!note] Аналіз складається з чотирьох етапів. Використовуйте чек-лист для відстеження прогресу:

- [ ]  **Перегляд метрик**: Оцінити ключові показники за вчорашній день.
- [ ]  **Виявлення аномалій**: Знайти відхилення від цільових значень (наприклад, ACoS >30%, CTR <0.3%).
- [ ]  **Аналіз причин**: Визначити, чому виникла аномалія (ключові слова, лістинг, конкуренція).
- [ ]  **Пропозиції рішень**: Запропонувати конкретні дії для виправлення.

### Кроки аналізу при роботі з товарами робіть по кроку:

1. **Відкрийте Dashboard у Amazon Advertising**:
    - Виберіть період "Вчора" і перегляньте метрики: Impressions, Clicks, Spend, Sales, ACoS, CTR, Conversion Rate.
2. **Сортування та виявлення аномалій**:
    - Сортуйте таблицю за ACoS (високий ACoS >30% вказує на неефективність).
    - Перевірте CTR (<0.3% може свідчити про нерелевантні ключові слова).
    - Оцініть Conversion Rate (<5% може вказувати на проблеми з лістингом).
3. **Аналіз причин**:
    - Перейдіть до **Search Term Report**, щоб перевірити, які ключові слова генерують кліки без конверсій.
    - Перегляньте лістинг товару (фото, опис, відгуки) для оцінки його якості.
    - Проаналізуйте дії конкурентів (знижки, кращі лістинги, нижчі ціни).
4. **Приклади пропозиції рішень**:
    - Для високого ACoS: Знизити ставки або додати негативні ключові слова.
    - Для низького CTR: Оптимізувати лістинг (наприклад, покращити фото чи заголовок).
    - Для низького Conversion Rate: Перевірити ціну або додати відгуки.

---
## 4. Ключові поради для PPC Manager’ів

- **Швидкість аналізу**: Щоденний аналіз має займати до 20 хвилин після ознайомлення з Dashboard (але може бути іншим, це залежить від кількості асінів).
- **Фокус на аномаліях**: Звертайте увагу на різкі зміни ( Аномалії наприклад, ACoS зріс із 20% до 50% ).
- **Перевірка причин**: Завжди аналізуйте Search Term Report і лістинг, щоб зрозуміти, чи проблема в кампанії чи зовнішніх факторах (конкуренти, ціна).
- **Конкретні пропозиції**: Уникайте загальних фраз, пропонуйте чіткі дії (наприклад, "Знизити бюджет кампанії A до 10 доларів / день").
- **Регулярність**: Щоденний аналіз допомагає уникнути накопичення проблем.
