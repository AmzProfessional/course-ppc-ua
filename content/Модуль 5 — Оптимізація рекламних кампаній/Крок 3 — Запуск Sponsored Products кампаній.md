## 1. Значення Sponsored Products кампаній

> [!note] Sponsored Products — це рекламні кампанії, які просувають окремі товари через ключові слова або таргетинг на продукти, підвищуючи видимість і продажі.

![[Pasted image 20250711192113.png]]

**Чому це важливо?**

- **Точний таргетинг**: Дозволяє показувати рекламу за релевантними ключами (exact, phrase, broad) або ASIN конкурентів.
- **Контроль витрат**: Налаштування ставок і плейсментів допомагають знижувати ACoS (<25%) і оптимізувати бюджет.
- **Простота запуску**: Ідеально для початківців і товарів із обмеженим запасом.
- **Прокачка органічної видачі**: Висока конверсія по ключам покращує органічні позиції.

**Приклад**: Кампанія Sponsored Products на ключ "baby bottle" (exact match, ставка 0.75) може дати Impressions = 5,000, Clicks = 50, Sales = 100 за 7 днів.

## 2. Процес запуску Sponsored Products кампанії

> [!note] Запуск кампанії включає вибір продукту, налаштування таргетингу, вибір ключів, встановлення ставок і плейсментів, а також перевірку SKU.

### Кроки створення кампанії:

1. **Перехід до створення**:
    - У Seller Central → Advertising → Campaign Manager → Create Campaign → Sponsored Products.
2. **Назва кампанії (Naming)**:
    - Використовуйте формат: "SP Keyword MatchType Key ASIN Test".
    - Приклад: "SP_Keyword_Exact_BabyBottle_ASIN123_Test".
3. **Вибір продукту**:
    - Введіть ASIN або SKU товару (наприклад, "SqueezeBottles346").
    - Перевірте статус SKU в Seller Central → Inventory → Manage Inventory (Available > 0).
    - **Важливо**: Уникайте вибору SKU з "Out of Stock", інакше кампанія не працюватиме.
4. **Тип таргетингу**:
    - Оберіть Manual Targeting → Keyword Targeting.
    - Видаліть автоматичні таргети за замовчуванням.
5. **Вибір ключів**:
    - Введіть ключ (наприклад, "baby bottle") у поле Keywords.
    - Виберіть тип відповідності (broad, phrase, exact). Для тесту почніть із broad.
    - Використовуйте Amazon Suggested Keywords для релевантних варіантів.
    - Приклад: Amazon пропонує "baby feeding bottle" (Suggested Bid = $ 0.84).
6. **Негативні ключі**:
    - Якщо ключ (наприклад, "baby bottle") уже є в exact match кампанії, додайте його до Negative Exact у broad match кампанії.
    - Приклад: Додайте "baby bottle" до Negative Exact, щоб уникнути дублювання.
7. **Ставки та плейсменти**:
    - Встановіть ставку (наприклад, $ 0.75, нижня межа Suggested Bid).
    - Стратегія: Down Only (рекомендується).
    - Плейсменти: Top of Search +50%, Rest of Search +0%.
8. **Бюджет**:
    - Встановіть $ 5–10/день для тестових кампаній.
    - Для дорогих товарів із високими ставками ($ 2–3) бюджет може бути більшим ($ 15+).
9. **Портфоліо**:
    - Виберіть портфоліо (наприклад, "SqueezeBottles") або залиште без портфоліо.
10. **Запуск**:
    - Перевірте назву, SKU, ключі, ставки, плейсменти.
    - Натисніть Launch Campaign.
    - Перевірте статус у Campaign Manager (Delivering після обробки).

### Чек-лист:

- [ ]  Створити кампанію з правильним Naming (SP_Keyword_Exact_BabyBottle_ASIN123).
- [ ]  Перевірити SKU (Available > 0).
- [ ]  Вибрати Manual Targeting → Keyword Targeting, видалити автоматичні таргети.
- [ ]  Додати 1–3 ключі (broad/phrase/exact) і негативні ключі (якщо є exact match кампанія).
- [ ]  Встановити Down Only, ставку ($ 0.75), плейсменти (Top of Search +50%).
- [ ]  Встановити бюджет $ 5–10/день, вибрати портфоліо.

## 3. Практичне завдання

#### **Завдання**: Запустіть тестову Sponsored Products кампанію для товару.

**Інструкція**:
1. Увійдіть у Seller Central → Advertising → Campaign Manager → Create Campaign → Sponsored Products.
2. Виберіть товар (наприклад, "пластикова дитяча пляшечка", ASIN123, SKU SqueezeBottles346).
3. **Налаштування**:
    - Назва: "SP_Keyword_Exact_BabyBottle_ASIN123_Test".
    - Перевірте SKU в Inventory → Manage Inventory (Available > 0).
    - Таргетинг: Manual → Keyword Targeting.
    - Додайте ключ "baby bottle" (exact match, ставка $ 0.75).
    - Додайте негативний ключ (наприклад, "sports bottle" до Negative Exact, якщо є broad match кампанія).
    - Стратегія: Down Only.
    - Плейсменти: Top of Search +50%, Rest of Search +0%.
    - Бюджет: $ 5/день.
    - Портфоліо: "SqueezeBottles" (або без портфоліо).
4. Запустіть кампанію, перевірте статус (Delivering) через 5–10 хвилин.
5. Запишіть результати:
    - Опис кампанії (назва, ключ, SKU, ставки, плейсменти).
    - Обґрунтування вибору ключа і налаштувань.

**Результат**:
- Опис: "Кампанія SP_Keyword_Exact_BabyBottle_ASIN123_Test, ключ ‘baby bottle’ (exact), SKU SqueezeBottles346, ставка 0.75, Down Only, Top of Search +50%, бюджет 5/день."
- Обґрунтування: "Ключ ‘baby bottle’ обрано через високу релевантність, exact match для контролю трафіку, Down Only для зниження CPC."

## 4. Ключові поради для Account Manager’ів

- **Перевіряйте SKU**: Завжди підтверджуйте, що SKU активний (Available > 0), щоб уникнути неактивних кампаній.
- **Починайте з exact**: Exact match забезпечує високу конверсію для релевантних ключів.
- **Використовуйте Suggested Bids**: Орієнтуйтеся на нижню межу пропозицій Amazon ( 0.50 – 1).
- **Контролюйте дублі**: Додавайте ключі з exact match до Negative Exact у broad/phrase кампаніях.
- **Моніторте статус**: Перевіряйте статус Delivering після запуску, щоб переконатися, що кампанія активна.
