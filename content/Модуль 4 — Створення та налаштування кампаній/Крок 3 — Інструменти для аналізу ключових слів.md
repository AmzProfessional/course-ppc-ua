## 1. Значення інструментів для аналізу ключових слів

> [!note] Інструменти дозволяють точно оцінити частотність, релевантність і конкурентність ключових слів, оптимізуючи вибір для PPC-кампаній і підвищуючи ефективність реклами.

#### **Чому це важливо?**

- **Helium 10**: Надає детальні дані про ключі конкурентів, частотність і релевантність, допомагаючи знаходити оптимальні запити.
- **Amazon Brand Analytics**: Показує реальні пошукові запити, кліки та продажі для товарів із історією продажів.
- **Product Opportunity Explorer**: Допомагає аналізувати ніші, сезонність і топові ключі для нових товарів.
- **Результат**: Використання інструментів зменшує ACoS, і допомагає "прокачати" органічні позиції.

**Приклад**: Для товару "пластикова дитяча пляшечка" Helium 10 Cerebro може знайти ключ "пляшечка без BPA" (Search Volume = 5,000, Sales = 100/тиждень), а Brand Analytics покаже, що він має Conversion Rate = 10%.

## 2. Огляд інструментів

### 2.1. Helium 10 — Cerebro

![[Pasted image 20250709181824.png]]

- **Що це**: Інструмент для аналізу ключових слів конкурентів за ASIN, показує частотність, релевантність і органічні позиції.
- **Як використовувати**:
    1. Увійдіть у Helium 10 → Tools → Cerebro.
    2. Введіть ASIN конкурента (наприклад, топовий товар за ключем "дитяча пляшечка").
    3. Увімкніть опцію "Exclude Variations", щоб аналізувати лише конкретний ASIN.
    4. Застосуйте фільтри:
        - Organic Rank: 1–20 (для релевантних ключів).
        - Keyword Sales: >10 продажів/тиждень.
        - Search Volume: >100 запитів/місяць.
    5. Сортуйте за Sales або Search Volume, щоб вибрати топові ключі.
- **Ключові метрики**:
    - **Organic Rank**: Позиція товару в органічній видачі.
    - **Keyword Sales**: Кількість продажів за ключем.
    - **Search Volume**: Частотність запитів.
    - **CPR (Clicks Per Rank)**: Кількість продажів за 8 днів для потрапляння на першу сторінку.
- **Приклад**: Cerebro показує ключ "пляшечка без BPA" (Organic Rank = 5, Sales = 100/тиждень, CPR = 80), який можна запустити в exact match.

### 2.2. Helium 10 — Magnet

![[Pasted image 20250826221442.png]]

- **Що це**: Інструмент для пошуку ключів за основним запитом, аналізу їх частотності та пов’язаних запитів.
- **Як використовувати**:
    1. У Helium 10 → Tools → Magnet → Find Suggestions.
    2. Введіть основний ключ (наприклад, "дитяча пляшечка").
    3. Фільтруйте за Sales (>10/тиждень) і Search Volume (>100/місяць).
    4. У вкладці Analyze Keywords додайте список ключів для детального аналізу (наприклад, "пляшечка без BPA", "дитяча пляшечка 200 мл").
- **Приклад**: Magnet показує, що ключ "дитяча пляшечка" має Search Volume = 10,000, Sales = 500/тиждень, але конкуренція висока.

### 2.3. Helium 10 — Keyword Tracker

![[Pasted image 20250826221555.png]]

- **Що це**: Інструмент для відстеження позицій за ключами для вашого товару та конкурентів.
- **Як використовувати**:
    1. У Helium 10 → Tools → Keyword Tracker.
    2. Додайте ASIN вашого товару та 5–10 ключів (наприклад, "дитяча пляшечка").
    3. Додайте ASIN конкурентів для порівняння.
    4. Аналізуйте зміни позицій (органічні та рекламні) щотижня.
- **Приклад**: Keyword Tracker показує, що ваш товар просів із 5-ї на 15-ту позицію за ключем "пляшечка для новонароджених", що сигналізує про необхідність коригування ставок.

### 2.4. Amazon Brand Analytics

- **Що це**: Внутрішній інструмент Seller Central для аналізу пошукових запитів, кліків і продажів (доступний для зареєстрованих брендів).
- **Як використовувати**:
    1. У Seller Central → Reports → Brand Analytics → Search Query Performance.
    2. Виберіть ASIN товару і період (наприклад, минулий місяць).
    3. Аналізуйте ключі за Search Volume, Impressions, Clicks, Sales і Conversion Rate.
    4. Виберіть ключі з високою конверсією (>5%) для запуску в PPC.
- **Приклад**: Brand Analytics показує, що ключ "пляшечка без BPA" має Conversion Rate = 10%, але Impressions = 1,000, що вказує на низьку видимість.

### 2.5. Product Opportunity Explorer

![[Pasted image 20250709182002.png]]

- **Що це**: Внутрішній інструмент Seller Central для аналізу ніш, сезонності та топових ключів.
- **Як використовувати**:
    1. У Seller Central → Growth → Product Opportunity Explorer.
    2. Введіть ASIN конкурента або основний ключ (наприклад, "дитяча пляшечка").
    3. У вкладці Niches перегляньте топові ключі, Search Volume і конкурентів.
    4. Аналізуйте сезонність у вкладці Trends (зростання/падіння попиту).
- **Приклад**: Product Opportunity Explorer показує нішу "Baby Bottles" із Search Volume = 50,000/місяць і ключем "пляшечка для годування" (Sales = 200/тиждень).

## 3. Ключові поради для PPC Manager’ів

- **Фільтруйте в Cerebro**: Використовуйте Organic Rank (1–20) і Sales (>10/тиждень) для вибору релевантних ключів.
- **Перевіряйте Magnet**: Аналізуйте пов’язані ключі для розширення списку.
- **Відстежуйте позиції**: Використовуйте Keyword Tracker для моніторингу змін у рангах.
- **Brand Analytics для історії**: Використовуйте для товарів із продажами, щоб знайти ключі з високою конверсією.
- **Сезонність у Product Opportunity**: Аналізуйте Trends для планування кампаній під сезонний попит.
