## 1. Значення пошуку ключових слів

> [!note] Пошук ключових слів дозволяє знайти релевантні запити, які забезпечують покази реклами вашого товару цільовій аудиторії, підвищуючи продажі та знижуючи ACoS.


![[keywords_tag_cloud.png]]

#### **Чому це важливо?**

- **Amazon Search**: Автодоповнення (Amazon Suggest) пропонує популярні запити, які відображають поведінку покупців.
- **Аналіз конкурентів**: Лістинги топових конкурентів містять ключові слова, які вже приносять продажі в ніші.
- **Баланс частотності та релевантності**: Високочастотні ключі дають більше показів, але можуть бути менш релевантними, тоді як низькочастотні ключі забезпечують вищу конверсію.
- **Прокачка органічної видачі**: Релевантні ключі з високою конверсією покращують органічні позиції товару.

**Приклад**: Для товару "пластикова дитяча пляшечка синя 200 мл" Amazon Suggest може запропонувати "дитяча пляшечка", "пляшечка для новонароджених", а лістинги конкурентів містять ключі типу "пляшечка для годування".

## 2. Методи пошуку ключових слів

> [!note] Основні методи пошуку включають використання Amazon Search (автодоповнення) і аналіз лістингів конкурентів для виявлення релевантних і частотних ключів.


![[HTH_1060x400_AccessibilityBanners_ProductAdvisor.jpg]]

### 2.1. Amazon Search (Автодоповнення)

- **Як працює**: Введіть основний запит (наприклад, "дитяча пляшечка") у пошуковий рядок Amazon, щоб отримати пропозиції автодоповнення.
- **Переваги**: Показує популярні запити, які вводять покупці, що відображають реальний попит.
- **Як використовувати**:
    - Введіть основний запит і запишіть 5–10 пропозицій автодоповнення.
    - Додайте уточнення (наприклад, "дитяча пляшечка синя") для пошуку більш релевантних ключів.
    - Приклад: "дитяча пляшечка" → "дитяча пляшечка для новонароджених", "пластикова дитяча пляшечка", "пляшечка 200 мл".

### 2.2. Аналіз конкурентів

- **Як працює**: Вивчіть лістинги топ-3 конкурентів за основним ключем, щоб знайти ключові слова в їх заголовках, bullet points і описах.
- **Переваги**: Дозволяє виявити ключі, які вже працюють для схожих товарів із високими продажами.
- **Як використовувати**:
    - Знайдіть топ-3 товари за основним ключем (наприклад, "дитяча пляшечка") у пошуковій видачі Amazon.
    - Проаналізуйте їх заголовки, bullet points і описи, виділивши ключові слова.
    - Приклад: Лістинг конкурента містить "пляшечка для годування без BPA" → додайте цей ключ до списку.

## 3. Процес відбору ключових слів

> [!note] Відбір ключів передбачає оцінку їх релевантності, частотності та конкурентності через аналіз видачі Amazon.

### Чек-лист відбору:

- [ ]  Зібрати 10–15 ключових слів через Amazon Search і лістинги конкурентів.
- [ ]  Оцінити релевантність: чи відповідає ключ характеристикам товару?
- [ ]  Оцінити частотність: перевірити кількість результатів у видачі Amazon (>50,000 — високочастотний, <5,000 — низькочастотний).
- [ ]  Оцінити конкурентність: перевірити топ-видачу за ключем (ціна, відгуки, рейтинг конкурентів).

### Кроки виконання:

1. **Збір ключів через Amazon Search**:
    - Введіть основний запит (наприклад, "дитяча пляшечка") у пошук Amazon.
    - Запишіть 5–7 пропозицій автодоповнення.
    - Додайте уточнення (наприклад, "дитяча пляшечка синя") і запишіть ще 3–5 пропозицій.
2. **Аналіз конкурентів**:
    - Знайдіть топ-3 товари за основним ключем.
    - Скопіюйте ключові слова з їх заголовків, bullet points і описів.
    - Приклад: Конкурент має заголовок "Пластикова дитяча пляшечка 200 мл без BPA" → ключі: "пляшечка без BPA", "дитяча пляшечка 200 мл".
3. **Оцінка ключів**:
    - Перевірте релевантність: чи відповідає ключ вашому товару (наприклад, "пляшечка синя 200 мл" — високо релевантний, "спортивна пляшка" — нерелевантний).
    - Оцініть частотність: введіть ключ у пошук Amazon і перевірте кількість результатів.
    - Оцініть конкурентність: порівняйте ціну, відгуки та рейтинг топ-3 товарів із вашим.
4. **Вибір ключів**:
    - Виберіть 3–5 ключів із високою/середньою частотністю і високою релевантністю для запуску в exact match.
    - Приклад: "дитяча пляшечка", "пляшечка для новонароджених", "пластикова дитяча пляшечка".

## 4. Ключові поради для PPC Manager’ів

- **Використовуйте автодоповнення**: Amazon Suggest відображає реальний попит покупців, що ідеально для пошуку частотних ключів.
- **Аналізуйте конкурентів**: Лістинги топ-3 товарів містять перевірені ключі, які вже приносять продажі.
- **Починайте з релевантних ключів**: Тестуйте високо релевантні ключі в exact match для високої конверсії.
- **Перевіряйте видачу**: Завжди оцінюйте топ-видачу, щоб переконатися, що ваш товар конкурентний за ключем.
- **Документуйте ключі**: Зберігайте список ключів із оцінкою релевантності та частотності для подальшого аналізу.
