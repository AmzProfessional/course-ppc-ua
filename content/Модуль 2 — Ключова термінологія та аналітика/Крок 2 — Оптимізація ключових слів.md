**Мета**: Навчитися оптимізувати ключові слова в PPC-кампаніях шляхом роботи з типами відповідності (broad, phrase, exact) та додавання негативних ключових слів для підвищення релевантності та ефективності.

> [!note] **Примітка**: Перегляньте відео, яке пояснює, як налаштовувати типи відповідності ключових слів і додавати негативні ключові слова в Amazon Advertising Console.

## 1. Значення оптимізації ключових слів

> [!note] Оптимізація ключових слів підвищує релевантність PPC-кампаній, зменшує марні витрати та покращує метрики, такі як ACoS, CTR і Conversion Rate.

**Чому це важливо?**

- **Типи відповідності (broad, phrase, exact)** визначають, наскільки точно ваші оголошення відповідають пошуковим запитам користувачів.
- **Негативні ключові слова** дозволяють виключити нерелевантні запити, що знижує витрати на неефективні кліки.
- Правильна комбінація типів відповідності та негативних ключових слів оптимізує бюджет і підвищує продажі.

## 2. Типи відповідності ключових слів

> [!note] Amazon Advertising підтримує три типи відповідності ключових слів, кожен із яких має свої особливості та сценарії використання.

### Огляд типів відповідності:

1. **Broad Match**:
    - **Що це?**: Оголошення показуються для широкого спектру запитів, включаючи синоніми, пов’язані слова та варіації (наприклад, "kitchen knife" може включати "chef knife" або "sharp knife").
    - **Коли використовувати?**: Для нових товарів або для збору даних про пошукові запити через Search Term Report.
    - **Плюси**: Максимізує охоплення, допомагає знайти нові релевантні запити.
    - **Мінуси**: Часто генерує нерелевантні кліки, що підвищує ACoS.
    - **Приклад**: Ключ "kitchen knife" може викликати покази для "knife set for kitchen".

2. **Phrase Match**:
    - **Що це?**: Оголошення показуються для запитів, які містять точну фразу або її близькі варіації (наприклад, "kitchen knife" може включати "sharp kitchen knife").
    - **Коли використовувати?**: Для більш точного таргетингу, коли ви хочете звузити аудиторію, але зберегти гнучкість.
    - **Плюси**: Баланс між охопленням і релевантністю.
    - **Мінуси**: Може включати частково нерелевантні запити.
    - **Приклад**: Ключ "kitchen knife" спрацює для "best kitchen knife" або "kitchen knife sharpener".

3. **Exact Match**:
    - **Що це?**: Оголошення показуються лише для точного запиту або його множинної форми (наприклад, "kitchen knife" або "kitchen knives").
    - **Коли використовувати?**: Для високоефективних запитів із хорошими продажами та низьким ACoS.
    - **Плюси**: Максимальна релевантність, низький ACoS.
    - **Мінуси**: Обмежене охоплення, може пропустити потенційні запити.
    - **Приклад**: Ключ "kitchen knife" спрацює лише для "kitchen knife" або "kitchen knives".

### Негативні ключові слова:

- **Що це?**: Запити, які ви виключаєте з таргетингу, щоб уникнути нерелевантних кліків.
- **Типи**:
    - **Negative Exact**: Виключає точний запит (наприклад, "toy knife").
    - **Negative Phrase**: Виключає запити, що містять фразу (наприклад, "cheap knife").
- **Коли використовувати?**: Для broad і phrase кампаній, а також для кампаній на конкурентів, щоб уникнути дублювання ключів із exact кампаній.
- **Приклад**: Додавання "toy knife" як негативного ключового слова для кампанії ножів, щоб уникнути показів для дитячих іграшок.

## 3. Процес оптимізації ключових слів

> [!note] Оптимізація ключових слів складається з трьох етапів. Використовуйте чек-лист для відстеження прогресу:

- [ ]  **Аналіз Search Term Report**: Виявити релевантні та нерелевантні запити.
- [ ]  **Додавання негативних ключових слів**: Виключити неефективні запити.
- [ ]  **Зміна типів відповідності**: Перенести релевантні запити в exact або phrase кампанії.

## 4. Практичне завдання

#### Як оптимізувати ключові слова для PPC-кампанії на основі Search Term Report.

**Інструкція**:
1. Увійдіть у Amazon Advertising і виберіть маркетплейс (наприклад, США).
2. Створіть Search Term Report для однієї кампанії (наприклад, Sponsored Products у broad match) за період "Останні 7 днів".
3. Завантажте звіт і відкрийте в Google Sheets або Excel.
4. Проаналізуйте дані:
    - Знайдіть один нерелевантний запит (наприклад, високий Spend, але нуль Sales).
    - Знайдіть один релевантний запит із хорошим ACoS (<30%) і Conversion Rate (>5%).
5. Виконайте оптимізацію:
    - Додайте нерелевантний запит до **Negative Keywords** у Campaign Manager (виберіть Negative Exact або Negative Phrase).
    - Створіть нову кампанію в **exact match** для релевантного запиту.
6. Запишіть дії та обґрунтування.

**Результат**:
- Опис кампанії (наприклад, "Sponsored Products, broad match для ASIN X").
- Нерелевантний запит (наприклад, "‘toy knife’: Spend = 15 доларів, Sales = 0").
- Релевантний запит (наприклад, "‘kitchen knife’: ACoS = 20%, Conversion Rate = 10%").
- Дії (наприклад, "Додано ‘toy knife’ до Negative Exact. Створено кампанію в exact match для ‘kitchen knife’ з бюджетом 10 доларів / день").

## 5. Ключові поради для PPC Account Manager’ів

- **Регулярність**: Перевіряйте Search Term Report щотижня, щоб вчасно додавати негативні ключові слова.
- **Точність**: Використовуйте exact match для перевірених ключів із високою конверсією.
- **Обережність із broad match**: Додавайте негативні ключові слова, щоб уникнути марних витрат.
- **Тестування**: Починайте з broad match для збору даних, потім переходьте до phrase і exact.
- **Якість лістингу**: Нерелевантні запити часто викликані слабким лістингом (перевірте фото, опис, ціну).
